## [1.0.0] - 2026-01-14

### Added
- Initial release of AI Vocab Builder skill
- Three vocabulary levels:
  - Basic Level (35 words) - Grade 1-5
  - Medium Level (35 words) - Grade 6-10
  - Matric Level (35 words) - Grade 11-12
- Daily learning system with 35 words per day
- Comprehensive daily testing system:
  - Multiple choice questions (5)
  - Fill-in-the-blank questions (5)
  - Sentence-based questions (5)
- Detailed results and feedback:
  - Score calculation
  - Words mastered tracking
  - Words to practice identification
  - Performance ratings
- Professional documentation:
  - README.md with full feature overview
  - Installation guide
  - Usage guide with examples
  - MIT License
- Multiple output formats:
  - Markdown (.md) files for editing
  - PDF files for printing
  - Date-organized file structure (YYYY-MM-DD format)
- Vocabulary features:
  - IPA pronunciation guides
  - Simple, clear meanings
  - 3 contextual examples per word
  - Usage frequency ratings
  - Subject/context area tags

### Initial Features
- ✓ Basic vocabulary learning
- ✓ Daily testing system
- ✓ Score tracking
- ✓ Progress feedback
- ✓ Multiple proficiency levels
- ✓ Professional documentation
- ✓ GitHub repository
